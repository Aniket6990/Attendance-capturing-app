
# BlueCore

"BlueCore" is a smart Attendance capturing app can be used to capture Attendance of Employees, Students and staffs.



# Tech Stack Used

 ## Frontend 
 - Flutter
 ## Backend
 - Nodejs
 - MongoDB
 - ExpressJS
 - Mongoose
